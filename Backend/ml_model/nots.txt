Model Server deployed/hostes on pythonAnywhere, just send a json object with keys
["CRIM", "NOX", "AGE", "TAX", "RM"] to the following link
les00.pythonanywhere.com/predict
use either a "POST" / "GET"

Returns a json object with key ["Prediction"] and value["HOUSE PRICE"(float)]

example

input = {
    "CRIM": 1.65660,
    "NOX": 0.71,
    "TAX": 403.0,
    "AGE": 10.3,
    "RM": 6
}


output = {
    "Prediction": 216822.35574302034
}
